## 📌 What is webintents-web?

webintents-web is a tool that adds an easy-to-use follow button for Fediverse platforms. The Fediverse is a group of social networks that work together but run on different servers. This app helps you connect and follow people across these networks with one click.

You don’t need to learn any technical steps to use it. webintents-web works as a universal button you can add to your website or use to follow others quickly. It makes joining and following on the Fediverse simpler and faster.
